ROLES DES LANGAGES
------------------

FRONTEND
--------
- HTML : Structure d'une page web (régions, blocs, médias ...)
- CSS  : Mise en forme (couleurs, tailles, bordures ...)
- JS   : Animation, intéractivité (slider, jeux, sécurité, expérience utilisateur ...)

BACKEND
-------
- PHP  : Fonctionnalités (connexion, inscription, panier, CRUD, sécurité, modèle de page). Java, Python, .NET ...
- SQL  : Communication avec les BDD (CRUD : Création, lecture, modification et suppression de données). SGBD (Système de Gestion de Base de Données : MySQL, PostgresSQL, NoSQL, Oracle, SQLServer ...)



3 Solutions pour faire un site web
----------------------------------
- From scratch : le dev fait tout à la main. Liberté totale mais temps extrèmement long.
- Framework : Laravel, Symfony, React, CakePHP ..., Liberté importante, temps de dev raccourci.
- CMS : Content Managing System (Drupal, Wordpress, Magento, Joomla, Prestashop ...). Beaucoup de contraintes liées au CMS choisi, temps de dev très court.
